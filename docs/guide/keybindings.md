# Keybindings Reference

## General

| Action           | Key | Description                                           |
| :--------------- | :---------- | :---------------------------------------------------- |
| Command Palette  |   `Ctrl+P` | Open the command palette to access all commands       |
| Show Keybindings |   `Ctrl+H` | Display all keybindings for your current configuration |


## File

| Action    | Key | Description                      |
| :-------- | :---------- | :------------------------------- |
| Open File |   `Ctrl+O` | Open a file from the file system |
| Save File |   `Ctrl+S` | Save the current file            |


## Editing

| Action                 | Key | Description                                    |
| :--------------------- | :---------- | :--------------------------------------------- |
| Undo                   |   `Ctrl+Z` | Undo the last change                           |
| Redo                   |   `Ctrl+Y` | Redo the previously undone change              |
| Select Next Occurrence |   `Ctrl+D` | Select the next occurrence of the current word |


## Navigation

| Action          | Key | Description                               |
| :-------------- | :---------- | :---------------------------------------- |
| Go to Definition | Command Palette | Navigate to the definition of a symbol    |
| Back            |      `Alt+Left` | Go back to the previous cursor position   |
| Forward         |     `Alt+Right` | Go forward to the next cursor position    |


## Layout

| Action           | Key | Description                              |
| :--------------- | :---------- | :--------------------------------------- |
| Split Horizontal |     `Alt+H` | Split the editor horizontally            |
| Split Vertical   |     `Alt+V` | Split the editor vertically              |
| Next Split       |     `Alt+O` | Move focus to the next split pane        |
| File Explorer    |   `Ctrl+E` | Toggle the file explorer sidebar         |


## Terminal

| Action                  | Key | Description                                  |
| :---------------------- | :---------- | :------------------------------------------- |
| Toggle Terminal Mode    | `Ctrl+Space` | Switch between terminal and editor mode      |
| Exit Terminal Mode      |     `Ctrl+]` | Exit terminal mode and return to editor      |
| Toggle Keyboard Capture |         `F9` | Toggle keyboard capture for terminal input   |
| Paste in Terminal       |     `Ctrl+V` | Paste clipboard contents into the terminal   |

Use the **Show Keybindings** (`Ctrl+H`) command within Fresh to see the full list of keybindings for your current configuration and OS.
