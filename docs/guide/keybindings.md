# Keybindings Reference

| Action | Key |
| :--- | :--- |
| **General** | |
| Command Palette | `Ctrl+P` |
| Show Keybindings | `Ctrl+H` |
| **File** | |
| Open File | `Ctrl+O` |
| Save File | `Ctrl+S` |
| **Editing** | |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |
| Select Next Occurrence | `Ctrl+D` |
| **Navigation** | |
| Go to Definition | Command Palette |
| Back | `Alt+Left` |
| Forward | `Alt+Right` |
| **Layout** | |
| Split Horizontal | `Alt+H` |
| Split Vertical | `Alt+V` |
| Next Split | `Alt+O` |
| File Explorer | `Ctrl+E` |
| **Terminal** | |
| Toggle Terminal Mode | `Ctrl+Space` |
| Exit Terminal Mode | `Ctrl+]` |
| Toggle Keyboard Capture | `F9` |
| Paste in Terminal | `Ctrl+V` |

Use the **Show Keybindings** (`Ctrl+H`) command within Fresh to see the full list of keybindings for your current configuration and OS.
